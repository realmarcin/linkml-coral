# CORAL LinkML Schema Diagrams

Entity-Relationship diagrams for the ENIGMA Common Data Model.

## Available Diagrams

- **CORAL Complete Schema** - Complete ENIGMA Common Data Model
  - [Mermaid](./full_schema.mmd)
  - [PNG](./full_schema.png)
  - [SVG](./full_schema.svg)

- **CORAL Schema Overview (No Attributes)** - High-level view of all entities without field details
  - [Mermaid](./schema_overview.mmd)
  - [PNG](./schema_overview.png)
  - [SVG](./schema_overview.svg)

## Viewing Options

1. **HTML Viewer**: Open `schema_visualization.html` in a web browser
2. **GitHub**: Mermaid diagrams (.mmd files) render automatically in GitHub
3. **VS Code**: Install the Mermaid extension to view diagrams
4. **Online**: Copy .mmd content to [mermaid.live](https://mermaid.live)
